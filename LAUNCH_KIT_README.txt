IMPORTANT SECURITY NOTES
================================================================================

✅ DO:
  - Store API keys in .env.local (never in code)
  - Use different keys for devnet vs. mainnet
  - Write seed phrases on paper & store in safe
  - Enable 2FA on all accounts
  - Rotate credentials every 90 days
  - Back up wallet JSON files securely

❌ DON'T:
  - Commit .env.local to GitHub
  - Share secret keys in messages or emails
  - Use same key for multiple services
  - Store keys in plain text files
  - Forget to add .env.local to .gitignore
  - Rush to mainnet without testing on devnet
