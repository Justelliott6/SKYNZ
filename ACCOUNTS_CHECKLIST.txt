PRIORITY 1: ESSENTIAL (Week 1)
================================================================================

[ ] GITHUB
    Link: https://github.com/signup
    Purpose: Version control & code hosting
    Setup:
      1. Go to https://github.com/signup
      2. Enter email, create password, username
      3. Verify email
      4. Clone SKYNZ repo: git clone https://github.com/Justelliott6/SKYNZ.git
    Key: Store GitHub PAT securely (Settings → Developer settings → Personal access tokens)

[ ] PHANTOM WALLET
    Link: https://phantom.app
    Purpose: Solana wallet for NFT minting & transactions
    Setup:
      1. Go to https://phantom.app
      2. Click "Download" → Install browser extension
      3. Create new wallet
      4. Save seed phrase (12 words) → WRITE ON PAPER & STORE SAFELY
      5. Set password
      6. Switch to Devnet (Settings → Network → Devnet)
      7. Request SOL airdrop: https://faucet.solana.com
    Key: NEVER share seed phrase or private key

[ ] ARWEAVE
    Link: https://www.arweave.org
    Purpose: Permanent NFT metadata storage
    Setup:
      1. Go to https://www.arweave.org
      2. Click "Get Started" → "Create Wallet"
      3. Download wallet file (JSON) → SAVE SECURELY
      4. Fund wallet with AR tokens (for upload fees)
      5. Use Bundlr for cheaper uploads: https://bundlr.network
    Key: Store wallet JSON file in secure location

[ ] AWS (Amazon Web Services)
    Link: https://aws.amazon.com
    Purpose: S3 bucket for media storage
    Setup:
      1. Go to https://aws.amazon.com
      2. Click "Create AWS Account"
      3. Enter email, password, account name
      4. Add payment method (credit card)
      5. Verify phone number
      6. Go to S3 → Create bucket (name: "skynz-media-prod")
      7. Generate access keys (IAM → Users → Security credentials)
      8. Store keys in .env.local (AWS_ACCESS_KEY_ID, AWS_SECRET_ACCESS_KEY)
    Key: NEVER commit AWS keys to GitHub
