## 🔑 API Keys & Credentials

### Create a `.env.local` file and add these keys:

```bash
# GitHub
GITHUB_TOKEN=your_github_pat_here

# Solana
SOLANA_RPC_URL=https://api.devnet.solana.com
SOLANA_NETWORK=devnet
SOLANA_WALLET_SECRET_KEY=your_wallet_secret_key_here

# Arweave
ARWEAVE_WALLET_KEY=your_arweave_key_here

# AWS S3
AWS_ACCESS_KEY_ID=your_aws_access_key
AWS_SECRET_ACCESS_KEY=your_aws_secret_key
AWS_REGION=us-east-1
AWS_S3_BUCKET=your-skynz-bucket

# Metaplex
METAPLEX_CANDY_MACHINE_ID=your_candy_machine_id
METAPLEX_CREATOR_WALLET=your_creator_wallet_address

# Magic Eden
MAGIC_EDEN_API_KEY=your_magic_eden_api_key

# Tensor
TENSOR_API_KEY=your_tensor_api_key

# Stripe
STRIPE_PUBLIC_KEY=pk_live_your_stripe_public_key
STRIPE_SECRET_KEY=sk_live_your_stripe_secret_key

# SendGrid
SENDGRID_API_KEY=your_sendgrid_api_key

# Database
DATABASE_URL=postgresql://user:password@localhost:5432/skynz

# App
APP_NAME=SKYNZ
APP_URL=https://skynz.io
NEXT_PUBLIC_APP_URL=https://skynz.io
```

### ⚠️ SECURITY NOTES:
- **NEVER commit `.env.local` to GitHub** (add to `.gitignore`)
- **NEVER share your secret keys** in messages or emails
- **Rotate keys regularly** (every 90 days)
- **Use different keys for devnet vs. mainnet**
- **Store backups securely** (password manager, encrypted drive)

## 🔐 Solana Wallet Setup

### Step 1: Create Phantom Wallet
1. Go to https://phantom.app
2. Click "Download"
3. Install browser extension
4. Create new wallet
5. **Save seed phrase securely** (write on paper, store in safe)

### Step 2: Get Devnet SOL
```bash
# Use Solana CLI to request airdrop
solana airdrop 2 <your_wallet_address> --url devnet

# Or use web faucet: https://faucet.solana.com
```

### Step 3: Connect to Devnet
1. Open Phantom wallet
2. Click "Settings" → "Network"
3. Select "Devnet"
4. Confirm you have SOL balance

### Step 4: Export Private Key (for backend)
1. Open Phantom wallet
2. Click "Settings" → "Security and privacy"
3. Click "Show Private Key"
4. Copy and store securely in `.env.local`

## 🎨 Metaplex Candy Machine Setup

### Step 1: Install Metaplex CLI
```bash
npm install -g @metaplex-foundation/sugar
```

### Step 2: Create Candy Machine Config
```bash
sugar create-config
```

### Step 3: Configure Settings
Edit `config.json`:
```json
{
  "label": "SKYNZ Founding Creator Pages",
  "symbol": "SKYNZ",
  "sellerFeeBasisPoints": 1000,
  "isMutable": true,
  "creators": [
    {
      "address": "your_wallet_address",
      "share": 100,
      "verified": true
    }
  ],
  "price": 0.5,
  "number": 100,
  "splToken": null,
  "gatekeeper": null,
  "endSettings": null,
  "hiddenSettings": null,
  "whitelistMintSettings": null,
  "itemsAvailable": 100
}
```

### Step 4: Upload Assets to Arweave
```bash
sugar upload ./assets
```

### Step 5: Deploy Candy Machine
```bash
sugar deploy
```

### Step 6: Verify Deployment
```bash
sugar show
```

## 🏪 Marketplace Integration

### Magic Eden API Integration
```bash
# Get API key from https://magiceden.io/developers
# Add to .env.local
MAGIC_EDEN_API_KEY=your_api_key

# List NFT
curl -X POST https://api-mainnet.magiceden.dev/v2/launchpad/collections \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "symbol": "SKYNZ",
    "name": "SKYNZ Creator Command Page",
    "description": "Creator-owned NFT page"
  }'
```

### Tensor API Integration
```bash
# Get API key from https://www.tensor.trade
# Add to .env.local
TENSOR_API_KEY=your_api_key

# Query marketplace data
curl https://api.tensor.trade/graphql \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "..."}'
```
